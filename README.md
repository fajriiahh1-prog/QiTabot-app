## 📖 Deskripsi Projek
**QiTabot** adalah aplikasi pembelajaran bahasa Arab berbasis web modern yang dirancang khusus untuk mengasah *Maharah Kalam* (keterampilan berbicara) bagi siswa tingkat Kelas X Madrasah Aliyah (MA). 

Aplikasi ini dibalut dengan desain antarmuka *Single-Page Layout* mewah bernuansa **Luxury Violet Velvet & Glamorous Gold**. Menggunakan framework **Streamlit**, QiTabot terintegrasi langsung dengan kecerdasan buatan **Google AI Studio (Gemini 2.5 Flash)** yang bertindak sebagai mentor virtual yang lemah lembut, sabar, cerdas, dan santun dalam mengevaluasi percakapan siswa.

### ✨ Fitur Utama:
* **Single-Page Premium Interface:** Mengusung desain terpusat di tengah halaman (tanpa menggunakan sidebar samping) dengan visual gradasi ungu violet yang elegan dan futuristik.
* **Sistem Akses Aman:** Autentikasi langsung di halaman utama menggunakan *Username* dan *API Key Google AI Studio*.
* **Mentor Pembimbing Pilihan:** Tersedia 2 pilihan karakter guru interaktif, yaitu **Ustadz Khalid** dan **Ustadzah Khaulah** yang membimbing secara kontekstual.
* **3 Kurikulum Esensial Kelas X MA:**
  1. *At-Ta'aruf* (Perkenalan / التعارف)
  2. *Al-Usroh* (Kehidupan Keluarga / الأسرة)
  3. *Al-Mihnah* (Profesi / المهنة)
* **Anti-Lag & Auto-Retry System:** Sistem tangguh yang otomatis mencoba mengirim ulang pesan jika server Google AI Studio sedang padat trafik (*high demand*).
* **Riwayat Percakapan Konten (Conversation History):** Mengingat seluruh konteks obrolan agar interaksi tanya-jawab berjalan mengalir secara natural.

## 🛠️ Prasyarat & Instalasi
Sebelum menjalankan aplikasi, pastikan perangkat komputer sudah terinstal Python (versi 3.10 ke atas sangat direkomendasikan).

### 1. Masuk ke Direktori Projek
Buka Command Prompt (CMD) Windows atau terminal pilihan Anda, lalu arahkan ke dalam folder proyek:
```bash
cd path/to/folder_projek

```

### 2. Instalasi Library Pendukung

Instal dua *library* utama yang dibutuhkan agar aplikasi berjalan dengan mengetik perintah berikut di CMD:

```bash
pip install streamlit google-genai

```

---

## 🚀 Cara Menjalankan Aplikasi

1. Pastikan file aplikasi utama (`app.py`) berada di dalam direktori folder tersebut.
2. Jalankan perintah Streamlit berikut melalui jendela CMD Anda:

```bash
   streamlit run app.py

```

3. Browser utama Anda (Chrome/Edge/Firefox) akan otomatis terbuka dan menampilkan halaman aplikasi pada alamat lokal: `http://localhost:8501`.

---

## 💡 Panduan Penggunaan Aplikasi

1. **Input Akun:** Pada kartu *Akses QiTabot Premium* di tengah halaman awal, masukkan **Username** dan **Google AI Studio API Key** valid milik Anda. Klik tombol **"Buka Akses Ruang Belajar ✨"**.
2. **Pilih Pengaturan:** Setelah akses terbuka, panel menu pilihan **Mentor Pembimbing** dan **Topik Pembelajaran** akan muncul sejajar di bagian atas halaman.
3. **Mulai Latihan Kalam:** Mentor virtual pilihan akan menyapa secara otomatis (*greeting*). Ketik respons balasan Anda menggunakan bahasa Arab (atau bahasa Indonesia) pada kolom *chat input* paling bawah.
4. **Interaksi & Evaluasi:** QiTabot akan merespons pesan lengkap dengan teks Arab berharakat, arti bahasa Indonesia, serta memberikan koreksi tata bahasa (*grammar*) yang santun jika terdapat kekeliruan kosakata.
5. **Tutup Sesi:** Jika ingin meriset obrolan, berganti akun, atau menyudahi sesi belajar, klik tombol merah **"Tutup Sesi Belajar ❌"** yang terletak tepat di bawah kolom input ketikan.
